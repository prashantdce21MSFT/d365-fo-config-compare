# Step 2 — Run onboarding

Onboarding will:

- Verify **Python 3.10+** is installed
- `pip install` runtime deps: `requests`, `msal`, `openpyxl`, `playwright`
- Check **Azure CLI** is installed (used for MCP backend)
- Register one or more **D365 environments** (label + base URL)
- Set the **output folder** for Excel extracts

You can rerun onboarding at any time.
